‚öôÔ∏è Request Parameters

Parameter Type Default Description
url* string - The URL of the website to screenshot
width number 1920 Viewport width in pixels
height number 1080 Viewport height in pixels
fullPage boolean false Capture the entire page length
format string png Image format: 'png' or 'jpeg'
waitStrategy string networkidle2 'networkidle0', 'networkidle2', 'domcontentloaded'
maxWaitTime number 10000 Max wait time in milliseconds
blockResources boolean false Block CSS, fonts, images for faster loading

* Required parameter
